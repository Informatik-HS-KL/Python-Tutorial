Erstellen Sie ein eigenes Modul mit den bereits erstellten Methoden aus den Aufgaben 
\ref{functionsAndModules/01_greet} - \ref{functionsAndModules/04_pythagoras}. 
Erweitern Sie das Modul um die Funktionen für die \emph{Summe} und das \emph{Produkt}. 
Beide sollen eine unbestimmte Anzahl von Parametern akzeptieren.

\begin{lstlisting}[language=Python]
# Neue mathematische Funktionen

def sum(...):
	... # addiert alle Zahlen zu einer Summe
	
def product(...):
	... # multipliziert alle Zahlen zu einem Produkt
\end{lstlisting}

Speichern Sie das Modul ab und verwenden Sie es in einem weiteren Programm. 

\begin{lstlisting}[language=Python]
# Nutzen des eigenen Moduls

import myModule

print(myModule.sum(
	myModule.product(2,3), myModule.product(3,5)))
# Ausgabe: 21
print(myModule.product(
	myModule.sum(2,3,4,5), myModule.product(0,9)))
# Ausgabe: 0

\end{lstlisting}
