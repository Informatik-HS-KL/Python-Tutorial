Wir beginnen mit einem \emph{Hello World} Beispiel:
\begin{lstlisting}[language=Python]
# Hallo Welt

print("Hello, World")
\end{lstlisting}
Schreiben Sie eine Funktion \emph{greet()}, die eine Gruﬂ-Nachricht in der Konsole ausgibt. 
Ohne Parameter soll immer \emph{Hello, World} ausgegeben werden, sonst wird der Wert der ‹bergabe verwendet:

\begin{lstlisting}[language=Python]
# Gruﬂ-Benachrichtigung

greet("Max Mustermann")
# Ausgabe: Hello, Max Mustermann
\end{lstlisting}
