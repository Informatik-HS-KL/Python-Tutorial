Der selbst erstellten Klasse muss ein Konstruktor hinzugefügt werden, der einen
Namen entgegen nimmt.
Dieser Name sollte in einem Attribut abgespeichert werden.
In der Schleife, welche in der \lstinline$run()$-Methode ausgeführt wird, muss nun zusätzlich der
Name ausgegeben werden.
Es müssen nun 100 Threads, denen unterschiedliche Namen übergeben werden,
erzeugt und gestartet werden.

Wird das Programm mehrmals ausgeführt, ändert sich die Reihenfolge, in der die Threads die
Ausgaben tätigen.
