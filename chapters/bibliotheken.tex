\chapter{Python Bibliotheken}
Zu Python gibt es eine ganze Reihe von sehr mächtigen Bibliotheken.
An erster Stelle sollen hier NumPy  und SciPy genannt werden.
Diese beiden Bibliotheken machen Python zu einem mächtigen Werkzeug
für numerische Berechnungen.
Statt numerischer Berechnung können wir auch symbolisch rechnen.
Hier fallen den Lesern sicher Software-Pakete wie \emph{Mathematica} oder \emph{Maple} ein, die
diese Art von Berechnungen schon sehr lange bietet.
Für den Einsatz in Python gibt es dazu die Bibliothek SymPy.
